Justificativa de Cada Teste

Teste 1 (testa_montagem_tabuleiro_corretamentee) => O teste em questão foi realizado para verificar
se a função esta criando a matriz que representa a entrada de forma correta, visto que essa matriz 
é de suma importancia para fazer as analises posteriores sobre a entrada.

Teste 2 (testa_pega_as_posicoes_das_rainhas_corretamente) => O teste em questão foi realizado com objetivo
de averiguar se a função esta pegando as posições corretas da rainhas que estão presentes no tabuleiro,
uma vez que essas posições serão essenciais para analisar as diagonais e retas das rainhas.

Teste 3 (testa_se_tabuleiro_tem_8_rainhas) => O teste em questão foi realizado com objetivo de conferir se a
função esta averiguando a presença das 8 rainhas no tabuleiro, uma vez que é necessario essa quantidade
especifica de rainhas para ser uma possivel solução do caso das Rainhas.

Teste 4 (testa_se_tabuleiro_tem_64_posicoes) => O teste em questão foi realizado para verificar
se a função esta averiguando a veriracidade do tabuleiro, uma vez que ele é de suma importancia para 
as analise futuras.

Teste 5 (testa_se_as_diagonais_da_rainha_escolhida_nao_tem_outras_rainhas) => O teste em questão foi realizado 
para verificar se a função esta checando as diagonais da rainha escolhida, a fim de atestar a ausencia de outras 
rainhas nelas.
